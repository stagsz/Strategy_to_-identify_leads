# CLAUDE.MD

## Project Overview

This project focuses on identifying and analyzing waste-to-energy condensation projects in Europe and the Nordic countries. It helps users discover potential leads and opportunities in this sector.

## Key Project Information

- **Project Name**: Strategy_to_-identify_leads
- **Focus Area**: Waste-to-energy condensation projects in Europe and Nordic countries
- **Primary Goal**: Identify and track viable projects for investment or partnership

## How Claude Can Help

When working on this project, focus on:

1. **Data Analysis**: Helping analyze and structure data about waste-to-energy projects
2. **Lead Identification**: Assisting in creating or improving strategies to identify leads
3. **Documentation**: Enhancing project documentation and research materials
4. **Development**: Building tools or features that support project discovery and tracking
5. **Research**: Gathering and organizing information about relevant projects

## Important Conventions

- All development should occur on feature branches prefixed with `claude/`
- Commit messages should be clear and descriptive
- Changes should be pushed regularly to keep the branch updated
- Code should follow existing patterns and conventions in the repository

## Data & Content Guidelines

- Focus on accurate, relevant information about waste-to-energy projects
- Ensure any external data sources are properly documented
- Keep research organized and traceable

## Development Workflow

1. Create or switch to the assigned feature branch
2. Make focused, well-documented changes
3. Test changes locally before committing
4. Write clear commit messages explaining the "why" behind changes
